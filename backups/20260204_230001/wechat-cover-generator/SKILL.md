# 微信公众号封面生成器

## 快速开始

```bash
cd .claude/skills/wechat-cover-generator

# 方法1：直接打开 HTML
# 双击 index.html 在浏览器中打开

# 方法2：启动本地服务
python -m http.server 8000
# 访问 http://localhost:8000
```

## 使用方法

1. 在浏览器中打开 `index.html`
2. 输入标题和副标题
3. 选择设计风格（5种预设）
4. 点击「导出封面」生成 PNG

## 设计风格

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| 极简格栅主义 | 黑白对比、秩序感、瑞士设计 | 技术文章、深度分析 |
| 弥散光感风 | 梦幻、治愈、流动 | 情感类、生活分享 |
| 孟菲斯波普风 | 活力、年轻、撞色 | 活动推广、热点话题 |
| 科技玻璃拟态 | 未来感、深度、透视 | AI 产品、科技资讯 |
| 东方新中式 | 雅致、文化、留白 | 文化类、国风内容 |

## 技术规格

- **比例**：3.35:1（900x269px 基础比例）
- **输出**：html2canvas 生成高清 PNG
- **依赖**：Tailwind CSS + Google Fonts (Noto Sans SC)

## 特点

- 实时预览，修改文字立即生效
- 多种预设风格一键切换
- 支持中文字体
