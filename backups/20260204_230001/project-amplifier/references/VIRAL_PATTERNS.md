# 病毒传播模式

## 四大黄金法则

### 1. 身份认同
让读者说："这就是我！"

### 2. 情绪放大
恐惧、好奇、愤怒、贪婪、渴望认可

### 3. 社交货币
转发能让我显得聪明/有品位/有见识

### 4. 实用价值
收藏起来下次用

## 高转化公式

```
痛点场景（3秒共鸣）
  ↓
情绪放大（你是对的）
  ↓
解决方案（我是答案）
  ↓
行动号召（现在就做）
```
