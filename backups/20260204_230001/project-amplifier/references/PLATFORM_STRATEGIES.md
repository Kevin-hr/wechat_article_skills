# 平台分发策略

## 抖音

| 维度 | 要求 |
|------|------|
| 时长 | 15-60秒 |
| 开头 | 3秒必须出痛点/金句 |
| 字幕 | 必须有，全屏大字 |
| 音乐 | 热门BGM带节奏 |
| 钩子 | "你应该知道的XX件事" |

## 小红书

| 维度 | 要求 |
|------|------|
| 封面 | 大字报+真人出镜/效果对比 |
| 标题 | 情绪化+数字："救命！XX太香了" |
| 正文 | 分点+emoji，阅读舒适 |
| 标签 | 精准+流量词 |

## B站

| 维度 | 要求 |
|------|------|
| 风格 | 深度+玩梗+真实感 |
| 开头 | "兄弟们" + 主题引入 |
| 长度 | 3-10分钟深度内容 |
| 结尾 | "一键三连" |

## 微信

| 维度 | 要求 |
|------|------|
| 公众号 | 1500-2500字，分段清晰 |
| 视频号 | 竖屏+文字弹幕 |
| 社群 | 私域承接+裂变诱饵 |

## 付费转化路径

```
公域流量（抖音/小红书）
    ↓
钩子内容（资料/工具/社群）
    ↓
私域沉淀（微信/社群）
    ↓
付费转化（产品/服务）
```
