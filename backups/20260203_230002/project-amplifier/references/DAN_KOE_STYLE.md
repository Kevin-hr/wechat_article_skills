# Dan Koe 写作风格

## 核心原则

1. **开头即钩子** - 前3秒必须抓住注意力
2. **第一人称直接对话** - 避免"用户应该"、"大家可以"
3. **短句+留白** - 每段不超过3行
4. **情绪共鸣优先** - 先说"我懂你"，再说"我有解法"
5. **CTA要狠** - 不要温和的"建议考虑"，要"现在就加我"

## 结构模板

### 公众号文章

```
【标题】情绪钩子 + 数字/反常识

【开头30字】
- 直击痛点
- "你是不是也..."

【正文3段】
- 场景还原（让读者看到自己）
- 情绪确认（你不是一个人）
- 解药呈现（我的方案）

【结尾】
- 情感升华
- 硬核CTA（加微信/扫码/关注）
```

### 短视频脚本

```
【0-3秒】痛点喊话
"你是不是也..."

【3-15秒】情绪放大
"每次XX的时候..."

【15-30秒】解决方案
"其实只需要..."

【30-45秒】证据/案例
"有人用这个方法..."

【45-60秒】行动
"点击下方/加我/关注"
```

## 禁用词

| 代替 | 用这个 |
|------|--------|
| 建议 | 你应该 |
| 可以考虑 | 直接 |
| 也许 | 就是 |
| 大家好 | （去掉，直接开始） |
