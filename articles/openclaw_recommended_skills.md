# OpenClaw 高质量技能推荐

OpenClaw 是一个运行在用户自有设备上的个人 AI 助手，支持多平台消息通道接入。与其他 AI 助手不同，OpenClaw 拥有一个庞大的"技能"（Skills）生态系统，通过 Skills 连接外部服务并执行复杂任务。这种模块化设计让 OpenClaw 具备了极强的扩展性——它不仅能聊天，还能帮你管理日程、操作浏览器、控制智能家居，甚至自动完成开发任务。

本文基于社区维护的 Awesome OpenClaw Skills 列表以及官方文档，为大家精选一批高质量且常用的技能。

## 一、核心必备能力

这些技能通常是系统内置或官方维护的，质量最高且最稳定。

### Web Search 与 Web Fetch

Web Search 允许 AI 搜索互联网并获取最新信息，Web Fetch 则能抓取网页内容。最新版本集成了 Brave Search API 和 Readability 提取功能，配合无头浏览器可以处理复杂的动态网页。这两个技能是 AI 了解实时信息的入口。

### Browser Control

通过 Chrome DevTools Protocol (CDP) 直接控制浏览器，OpenClaw 能够像真人一样点击、滚动、填写表单，甚至处理复杂的网页交互。比如让它帮你订餐、填写申请表，或者从网页中提取特定数据。

### Terminal Execution

允许 AI 运行本地 Shell 命令，这是 OpenClaw 被称为"开发者助手"的基础。它可以用来安装软件、运行脚本、管理进程，或者执行任何系统命令。配合代码审核工具，AI 还能帮你审查代码质量和安全性。

## 二、生产力与办公

对于日常办公场景，OpenClaw 的技能生态覆盖了主流办公工具。

### Google Workspace / gog

这个技能可以管理 Gmail 邮件、日历事件和 Google Drive 文档。想象一下：让 AI 自动清理一周前的旧邮件，或者根据邮件内容安排日程，甚至总结长文档的要点。它极大降低了在多个应用间切换的成本。

### Notion 与 Obsidian

Notion 技能支持全功能的 CRUD 操作，可以管理数据库和页面，适合作为项目文档中心。Obsidian 技能则专注于读取和管理本地 Markdown 笔记库，帮你打造"第二大脑"。两个技能配合使用，可以实现云端与本地笔记的双向同步。

### Linear 与 Jira

这两个项目管理工具的技能深受开发者喜爱。Linear 技能可以查询 Issue、创建任务、更新状态，而 Jira 技能则支持更复杂的工作流配置。让 AI 管理 Bug 跟踪，能让开发者专注于编码本身。

## 三、开发与编码

OpenClaw 对开发者非常友好，大量开发工具的集成让 AI 成为真正的编程助手。

### GitHub

通过集成 gh CLI 工具，AI 可以管理仓库、查看 Pull Request、审查代码，甚至自动合并已通过 CI 的代码。你可以让它帮你写 Commit Message，或者总结某次代码变更的内容。

### Vercel 与 Cloudflare

Vercel 技能支持一键部署应用、查看部署状态、回滚版本。Cloudflare 技能则可以管理 DNS 记录和边缘函数。对于全栈开发者来说，这两个技能让应用部署变得像说话一样简单。

### Coding Agents

OpenClaw 还支持集成 claude-code 或 cursor-agent 等专业编程代理。当遇到复杂的重构任务时，可以调用这些专门的编程代理来处理，OpenClaw 本身则负责协调和整合结果。

## 四、搜索与研究

对于需要深度研究的任务，OpenClaw 的研究类技能表现出色。

### Perplexity 与 Exa

相比普通搜索，这两个技能提供更精准的、面向 AI 优化的搜索结果和引用。Perplexity 会给出带来源的答案，Exa 则擅长语义搜索和内容归档。它们是撰写技术文档时的得力助手。

### YouTube Summarizer

获取 YouTube 视频字幕并生成结构化摘要，再也不用花几十分钟看一个教程视频了。让 AI 提取关键步骤，直接跳转到对应时间点即可。

### ArXiv Watcher

自动监控和总结最新的学术论文。对于研究者来说，这个技能可以帮你追踪特定领域的最新进展，把论文摘要整理成每日简报。

## 五、生活与硬件控制

OpenClaw 不仅能干活，还能帮你管理生活。

### Home Assistant

控制家中的智能设备，包括灯光、开关、传感器等。有用户用它来根据空气质量传感器自动控制净化器，或者设置离家/回家场景的自动化。OpenClaw 接入后，你可以用自然语言控制整个智能家居。

### Spotify 与 Apple Music

控制音乐播放、管理播放列表、发现新音乐。早上想让 AI 播放提神的音乐，直接用语音告诉它就行。

### Health（Whoop/Oura/Apple Health）

读取健康数据，包括睡眠质量、心率变异性、每日活动量。AI 可以基于这些数据提供每日健康简报，甚至给出改善建议。

## 六、安全警告

虽然 ClawHub（技能商店）提供了大量技能，但安装时务必谨慎。

**供应链风险**：安全研究人员曾演示过，攻击者可以通过刷高下载量来推广包含恶意代码的技能。

**权限过大**：安装的技能通常继承 AI 的权限。如果技能包含恶意代码，它可能读取您的 SSH 密钥或敏感文件。

**建议**：尽量使用官方推荐的技能，或者在安装第三方技能前检查其源代码，路径在 `~/.openclaw/skills/` 目录。

## 如何安装技能

通过 ClawHub 可以快速安装技能。安装前请仔细阅读技能文档和用户评价，避免安装来源不明的技能。

```bash
# 使用 ClawHub 安装（需谨慎）
npx clawhub install <skill-name>

# 或者让 AI 自己写技能
# "帮我写一个查询币安比特币价格的 skill" -> 它会自动生成并测试代码
```

OpenClaw 的技能生态让它不仅仅是一个聊天机器人，而是一个真正能帮你完成实事的 AI 助手。从开发调试到生活管理，从信息检索到硬件控制，技能的扩展让边界不断延伸。选择适合自己场景的技能，OpenClaw 会成为你的全能数字伙伴。
