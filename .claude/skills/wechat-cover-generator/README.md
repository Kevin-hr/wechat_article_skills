# WeChat Cover Generator

基于 HTML/CSS 的微信公众号封面生成器，支持多种设计风格一键切换。

## 特性

- **严格比例**：整体 3.35:1，完美适配公众号列表 + 朋友圈分享
- **多种风格**：极简格栅、弥散光感、孟菲斯波普、科技玻璃、东方新中式
- **一键下载**：使用 html2canvas 生成高清 PNG
- **实时预览**：修改文字立即生效

## 使用方法

### 直接打开 HTML

双击 `index.html` 在浏览器中打开即可使用。

### Python 启动本地服务

```bash
cd wechat-cover-generator
python -m http.server 8000
```
访问 `http://localhost:8000`

## 设计风格库

### Style A: 极简格栅主义 (Minimalist Grid) [默认]
- **核心**：黑白对比、秩序感、瑞士平面设计
- **配色**：背景纯黑 (`#1a1a1a`)，文字纯白，点缀荧光绿 (`#00ff00`) 或克莱因蓝
- **装饰**：细线条箭头、数字编号、十字准星

### Style B: 弥散光感风 (Gradient Blur)
- **核心**：梦幻、治愈、流动
- **配色**：蓝紫、橙粉渐变背景
- **视觉**：高斯模糊毛玻璃效果

### Style C: 孟菲斯波普风 (Memphis Pop)
- **核心**：活力、年轻、冲突
- **配色**：高饱和度撞色（黄配紫、红配蓝）
- **视觉**：粗黑边框、硬阴影

### Style D: 科技玻璃拟态 (Tech Glass)
- **核心**：未来感、深度、透视
- **配色**：深蓝/深紫渐变，赛博朋克霓虹点缀
- **视觉**：半透明玻璃层、发光网格线

### Style E: 东方新中式 (New Chinese)
- **核心**：雅致、文化、留白
- **配色**：中国传统色（天青、朱红、墨黑）
- **视觉**：竖排文字、印章效果

## 技术栈

- HTML5 + Tailwind CSS (CDN)
- html2canvas (CDN)
- Google Fonts (Noto Sans SC)
