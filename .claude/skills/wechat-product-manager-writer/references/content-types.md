# 五类内容方向

| 类型 | 触发词 | 写作角度 | 配色 |
|------|--------|---------|------|
| AI 产品拆解 | "分析"、"拆解" | 真实使用场景 + 产品设计逻辑 | 蓝紫渐变 |
| 场景解决方案 | "怎么用 AI 做 XXX" | 具体实现步骤 + 效果和坑 | 绿橙渐变 |
| 效率提升实战 | "技巧"、"心得" | 对比前后效果 + 配置分享 | 橙黄渐变 |
| 产品方法论 | "如何"、"为什么" | 观点 + 论证 + 落地方法 | 深蓝渐变 |
| 行业观察 | "怎么看"、"趋势" | 鲜明观点 + 实际观察支撑 | 蓝绿渐变 |
